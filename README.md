# Geometria 3D Interativa 🧊

[![Live Demo](https://img.shields.io/badge/Acessar-Projeto_Online-082f49?style=for-the-badge&logo=github)](https://jeiversonchristian.github.io/Geometria-3D-Interativa/)

**Geometria 3D Interativa** é uma ferramenta educacional visual e interativa desenhada para apoiar aulas de matemática na educação básica, focando no estudo do eixo temático de **Grandezas e Medidas**. O projeto permite a manipulação dinâmica de figuras tridimensionais (Cubo, Bloco Retangular e Cilindro), facilitando a compreensão de conceitos como vértices, arestas, faces, cálculos de Área da Base e Volume.

🔗 **Acesso ao projeto ao vivo:** [https://jeiversonchristian.github.io/Geometria-3D-Interativa/](https://jeiversonchristian.github.io/Geometria-3D-Interativa/)

---

## 🚀 Funcionalidades Principais

- **Visualização 3D Dinâmica:** Motores gráficos renderizados via `Three.js` com suporte a câmera ortográfica para manter as proporções estritas das linhas paralelas.
- **Interação Total:** Rotação e Zoom acessíveis por botões na interface ou comandos livres pelo mouse/touch.
- **Controle de Parâmetros e Cálculos:** Inputs em tempo real que alteram as arestas (C, L, H) e o raio (R), resolvendo automaticamente os cálculos de Área da Base (Ab) e Volume (V) das figuras.
- **Customização Visual Pedagógica:**
  - Alteração de cor global e de faces/superfícies de forma independente.
  - Ocultar ou destacar vértices com controle preciso de tamanho.
  - Textos `C`, `L`, `H`, `R` sobrepostos e alinhados dinamicamente com as arestas no ambiente 3D.
  - Visualização de raio físico e origem no cilindro.
- **Tema Escuro (Dark Mode):** Interface moderna que destaca formas e reduz a fadiga visual, desenhada com CSS puro e responsivo para celulares, tablets ou TVs de sala de aula.

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3** (Vanilla, sem frameworks de estilização para garantir performance e customização direta).
- **JavaScript (ES6+)**
- **Three.js** (WebGL 3D Engine) + OrbitControls
- **CSS2DRenderer** (Módulo oficial do Three.js para renderização de rótulos HTML acompanhando coordenadas 3D estritas).
